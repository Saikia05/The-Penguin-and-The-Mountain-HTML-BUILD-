# The Penguin and the Mountain

A relaxing single-file HTML5 isometric exploration + mindfulness mini-game.

## Description

You play as a little penguin wandering through a dreamy volcanic-ice landscape filled with glowing crystals, hot springs, auroras, and gentle snow.  
The journey leads to two peaceful caves containing over 15 calming mini-activities designed to help you breathe, focus, create, and unwind.

**No timers. No fail states. Just gentle presence.**

## Features

- Procedurally lit isometric world with day/night aurora cycle  
- Smooth penguin movement & camera follow  
- 2 explorable caves with 15+ relaxing mini-games / activities  
- Mindfulness-oriented mini-puzzles:  
  • Breathing exercise  
  • Zen sand raking  
  • Gratitude journal  
  • Bubble popping  
  • Memory cards  
  • Pattern memory  
  • Free doodle & coloring  
  • Drum pad  
  • Maze escape  
  • Balloon popping  
  • Number grid logic  
  • …and more  
- Pure JavaScript + Canvas (no external libraries)  
- Procedural audio (Web Audio API)  
- Keyboard + partial mouse/touch support

## How to Play

1. Open `erebus_v12_passA.html` in any modern browser  
2. Click **Start** on the title screen  
3. Use **WASD / Arrow keys** to move  
4. Press **E** near glowing passages or NPCs to interact  
5. Explore the mountain and enter the caves for relaxing activities

## Controls

- **WASD / Arrows** → Move  
- **E / Space / Enter** → Interact / advance dialogue / skip intro  
- **Mouse / Touch** → Most mini-games & UI  
- **Esc** (in some browsers) → may help exit full-screen

## Tech Highlights

- Single HTML file (~3.6 MB with embedded background)  
- Isometric tile rendering with height-based lighting  
- Lazy-initialized Web Audio for all sounds  
- No save system — designed as a short, stateless relaxation experience
